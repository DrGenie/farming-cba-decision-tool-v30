Help and frequently asked questions

Why are results missing?

Run the analysis after loading data. Also check that treatment names, yield values, and total farm costs per hectare are present and numeric.

Why is the ranking table empty?

The ranking table fills only after a successful analysis. Go back to the Data tab, check the validation messages, and then run the analysis again.

How should the workbook be structured?

Use the official template. Keep the header row, enter one row per replicate, and keep the fields Amendment Name, Crop_yield_t_ha, and Total Farm Costs_per_ha populated.

What if I want to compare two non-control treatments?

Go to Results, switch the comparison mode to treatment versus treatment, and select two different treatments.

What if upload does not work?

Use a modern browser and keep the internet connection active so the Excel reader library can load. The website opens with the provided trial workbook already loaded, so you can still use the tool immediately.

User guide

Use the Data tab to confirm the workbook and preview the rows, the Assumptions tab to set prices and discounting, the Results tab to compare treatments, and the Sensitivity tab to test alternative numeric assumptions.
